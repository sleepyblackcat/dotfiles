##tmux
###Quick Start
```
# install the Tmux Plugin Manager
$ git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
# copy config file 
$ cp .tmux.conf ~/
# open tmux
# press Ctl-a I to install plugins
# press Ctl-a R to resource the config
```
